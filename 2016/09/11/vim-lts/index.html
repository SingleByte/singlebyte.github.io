VIM-LTS

都说vim是神器，以前都是在服务器上编辑文件用一用，最近闲来，准备坚持使用，看看是否能够真正提升开发效率。这篇文章也会记录这次尝试的。希望一切顺利！

基础篇

环境搭建

我是idea的忠实粉丝，就像google抛弃eclipse转向idea一样，优秀是一种气质。

idea配置vim插件很简单：

+ plugin里面直接下载IdeaVim
+ 重启idea
+ 默认 ctrl+alt+v 开关

开启我们的vim旅程吧….

vim模式

+ insert模式
+ normal模式

5个基本命令切记

+ i     进入insert模式，esc切入命令模式  
+ x     删除光标所在的那一个字符
+ ：wq   保存退出
+ dd    删除当前行，并且存入剪切板
+ p     粘贴剪贴板的内容

熟记方向键减少右手活动范围:hjkl (←↓↑→)

进阶篇

高级进入insert模式

+ a     在光标后插入（本人现在还未使用过，感觉相对于i来说的）
+ o     在当前行后插入新的一行（就是回车换行进入insert模式）
+ O     在当前行前插入新的一行 （水平不够，没用过）
+ cw    把光标所在位置的一个单词删掉进入insert（这个感觉很强大，没用过，以后知道什么场景会用，来更新）
